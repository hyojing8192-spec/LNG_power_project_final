## 4. 핵심 설계 결정 사항

### 4-1. SMP 이상 임계값 — 동적 역산 방식 (F4.0)

기존의 고정 기준값(100원, 170원) 대신, **이번 달 LNG가격·환율·열량으로 BEP를 역산**하여
그 달의 실제 경제성 한계에 맞는 임계값을 동적으로 산출한다.

**역산 공식:**

```
대체단가_BEP = (LNG가격 + 제세금) × 환율 × 52(MMBtu/ton) / (1293(Nm³/ton) × 열량) × 효율
```

| 구분 | smp_low (LNG발전 2기) | smp_high (기력발전) |
|------|----------------------|-------------------|
| 사용단가 (제세금 0) | ← LNG발전 2기 효율 1.575 Mcal/kWh | ← 기력발전 효율 2.3 Mcal/kWh |
| Spot LNG (제세금 0.8) | 사용단가 대비 높게 산출 | 사용단가 대비 높게 산출 |

예시 (LNG 11$/MMBtu, 환율 1380, 열량 9.107):

| | smp_low | smp_high |
|---|---|---|
| 사용단가 | **105.6원/kWh** | **154.2원/kWh** |
| Spot LNG | **113.3원/kWh** | **165.4원/kWh** |

> 기존 고정값 100원·170원은 폴백(fallback)으로만 유지.

---

### 4-2. 기력발전 BEP 계산 (F3.3-S)

```python
# 사용단가 기준 (제세금 0)
BEP = (대체단가 / 효율) × 열량 × 1293 / 52 / 환율

# Spot LNG 기준 (제세금 0.8)
BEP = (대체단가 / 효율) × 열량 × 1293 / 52 / 환율 - 0.8
```

- **효율**: 2.3 Mcal/kWh (기력발전 기본값)
- **대체단가**: SMP ≥ smp_high 구간에서 역송 100% 근사 → 대체단가 ≈ SMP
- **가동 가능 조건**: `steam_BEP > LNG가격`

---

### 4-3. 이상치 3단계 분류 체계

| anomaly_type | 조건 | 권고 운전 방향 |
|---|---|---|
| `SMP 제로` | SMP ≤ 0 | LNG발전 즉시 감발/정지 검토 |
| `SMP 경제성 한계` | 0 < SMP < smp_low | 1기 또는 저부하 운전으로 전환 검토 |
| `SMP 과대` | SMP ≥ smp_high | 기력발전 LNG 점화 추가 검토 |

---

### 4-4. LNG 가격 유형별 제세금 처리

| 유형 | 설명 | 제세금 |
|------|------|--------|
| 사용단가 | 이미 도입·정산 예정인 계약 LNG | 0 (가격에 포함) |
| Spot LNG | 시장에서 추가 구매하는 현물 LNG | 0.8 $/MMBtu |

Streamlit UI에서 `is_spot` 토글로 선택하며, 선택에 따라 BEP 계산과 SMP 임계값이 모두 연동 변경된다.
