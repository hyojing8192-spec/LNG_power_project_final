## 문제 인식

> **핵심 질문:** “어떤 문제가, 왜 발생하고 있는가?”
> 

### **문제 정의**

- 에너지 수급담당자는 매일 smp(역송단가)를 직접 확인하며 LNG발전 경제적 가동패턴을 만들고 수립하고있는데, 국가 내 재생에너지 확대로 인한 역송단가 지속 하락으로 smp가 0으로 떨어지는 구간이 존재하며, 170이상으로 뛰는 구간도 발생. 광양소는 smp값이 평균 100원/kWh이하로 하락 시 LNG발전 역송 경제성 확보가 불가한데 매일 수급담당자가 수작업으로 확인하기때문에 놓칠 수도 있는상황이 있음

170원/kWh 이상으로 증가했을때는 LNG발전 + 기력발전 LNG점화까지 추가해서 돈을벌어야하고
기준치 이하로 떨어지면 LNG발전 가동패턴을 바꿔야함.

### **주요 업무 흐름**

1. 매일 아침, 저녁으로 에너지 수급담당자가 역송단가 확인
2. 시간대별 역송단가를 바탕으로 LNG발전 가동경제성 파악(경제성 파악 시 활용 변수 : smp, 수전단가,LNG발전 효율, LNG가격, 환율 등)
3. 경제성 파악에 따른 LNG발전 가동 계획 수립
4. 가동계획 현장 전파

### LNG발전 경제성 판단시 참고사항
LNG발전량이 수전량,역송량을 얼마나 대체하는지 대체단가를 산정
(ex. LNG발전 2기 가동 시 부하 약 570MW, 역송 300MW일때, 270MW는 수전대체인데
1기 가동은 570MW의 절반인 285MW수준이므로, 1기가동은 거의 수전대체
1기 추가가동은 거의 역송대체로 판단.
1기 대체단가 : 285MW 중 270MW는 수전대체(270/280), 15MW는 역송대체(15/280)
추가 1기 대체단가 : 285MW 전량 역송대체

2기 저부하 운전 시(380000~410000kW) 수준으로, 수전 Zero화 역송 Zero화 운전을 수행
근데 LNG발전 효율이 1.68~1.69Mcal/kWh 수준으로 하락해서
경제성이 떨어지는 부분이 있음.

대체단가를 산정하면 이 단가에 따른 LNG발전 가동 BEP를 산정
BEP 산정식 : LNG발전 대체단가(원/kWh) / LNG발전효율(Mcal/kWh) * LNG열량(Mcal/N㎥) * (1293N㎥/ton) / (52MMBtu/ton) / 환율(원/$) - 제세금(0.8$/MMBtu)

LNG가격이 BEP보다 낮으면 구매해서 LNG발전을 가동하는게 이득

보통 LNG발전 2기 가동을 하게되면 발전량은 530000~595000kW 까지 왔다갔다함.
계절별 특성이있어서 하절기에는 발전효율이 좀 떨어지고 출력도 낮아지는데
겨울철에는 발전효율이 좋아지고 출력도 증가

LNG발전 1기 가동을 하게되면 2기가동의 절반수준임, 이때의 수전량은 보통 평균 150000kW

LNG발전 2기 저부하 운전은 380000~410000kW 수준, 이때의 역송, 수전량은 100000kW 이내

역송, 수전량은 그때그때 제철소 전력부하에 따라 다르지만
보통 고로가 수리하는날은 LNG발전 2기 가동을 해도 수전이 발생하고 약 100000kW 수준,
열연 수리 또는 냉연/도금 수리 겹치는날은 역송이 300000kW 까지 발생함.

고로수리는 한달에 하루~이틀정도 있음. 열연 대수리는 4개의 열연이 1년중에 평균 7일정도 진행하고
대수리 시, 한 열연당 50000~60000kW정도 부하가 줄어들기때문에 그만큼 역송량도 늘어남.
그 수준이 약 280000~300000kW 수준이며, 이는 열연 대수리에 다른 설비들 수리까지 겹치기때문에 늘어나는 역송량이므로 열연 대수리가 없는 평시에는 최대 150000~250000kW 수준으로 역송이 발생.

LNG발전 가동패턴은 크게 2기 full 가동, 1기 full 가동, 2기 저부하 운전이 있음

역송단가가 낮아져서 대체단가가 낮아지면 경제성이 떨어지기때문에 LNG발전 운전 형식을 1기 또는 2기 저부하로 변경할 필요가있고,
역송단가가 높아져서 대체단가가 높아지면, LNG발전이 1기 가동하고있었다면 2기 저부하 또는 2기가동으로, 2기 저부하를 하고있었다면 2기 가동으로 발전량을 증대할 필요가 있음.
추가로 역송단가가 170원/kWh 이상으로 올라가면 기력발전에도 LNG를 추가 점화하여 가동할 필요가 있음.

여기서 기력발전 가동 경제성은 보통 2.3Mcal/kWh 기준으로 LNG발전 BEP 산정하는 방식과 동일
BEP 산정식 : LNG발전 대체단가(원/kWh) / 기력발전발전효율(Mcal/kWh) * LNG열량(Mcal/N㎥) * (1293N㎥/ton) / (52MMBtu/ton) / 환율(원/$) - 제세금(0.8$/MMBtu)

여기서 LNG가격은 매달마다 정할 수 있는데
사용단가 기준, Spot 기준에 따라 BEP 계산식도 달라짐.

사용단가 기준 :
BEP 산정식 : LNG발전 대체단가(원/kWh) / 기력발전발전효율(Mcal/kWh) * LNG열량(Mcal/N㎥) * (1293N㎥/ton) / (52MMBtu/ton) / 환율(원/$)

Spot LNG 기준 :
BEP 산정식 : LNG발전 대체단가(원/kWh) / 기력발전발전효율(Mcal/kWh) * LNG열량(Mcal/N㎥) * (1293N㎥/ton) / (52MMBtu/ton) / 환율(원/$) - 제세금(0.8$/MMBtu)

사용단가는 이미 도입되어 해당월에 정산될 LNG 가격이라면, 
Spot LNG는 아직 구입하지않은 시장가이기때문에 도입 시 세금이 붙음.
그래서 BEP 계산 시 제세금이 고려돼야함.

먼저 SMP 이상치 탐지의 기준은 해당월에 적용하는 LNG가격이 돼야할 것 같아. 예를 들어 11$/MMBtu라면 BEP계산식을 역순으로 적용하면 전력단가가 나오니 그에 해당하는 값을 기준으로 설정해야해. 사용단가, Spot LNG일때를 나눠서 봐야할 것 같고

한전 수전단가 파일을 가지고 수전단가는 집어넣어야함.

LNG열량은 학습데이터 평균값을 활용하고
환율데이터는 전일 평균으로 활용하면 되는데,
전일 평균값은 매일 새로 끌어와야함.
예를들어 내일 LNG발전 가동 패턴을 판단하기 위해서는 오늘 환율 데이터를 끌어와야함.

경제성 금액 산정 방식 
: (LNG발전 BEP - LNG 가격)($/MMBtu) * LNG발전량(kW) * LNG발전효율(Mcal/kWh) / LNG열량(Mcal/N㎥) / 1293(N㎥/ton) * 52(MMBtu/ton) * 환율(원/$) / 100000000

2기저부하 운전 : 데이터 불균형 처리 필요 가우시안모형, smote로 처리필요

LNG발전 BEP를 구해서 LNG가격이랑 비교하는 걸로 가동판단을 하는 로직으로 구성해야해. LNG발전 BEP가 LNG가격보다 크면
가동하고, LNG발전 BEP가 LNG가격보다 낮으면 가동하면 손해

그리고 경제성을 볼때에는 1기 가동이랑 2기 가동을 따로 비교하는게 아니고, 1기 가동을하고 추가로 1기를 가동하는거기때문에 2기 가동했을때에 경제성이 있다고
하면 무조건 2기를 가동하는게 맞아. 1기 가동하는거에 추가로 돈을 더 버는거라 그렇게 가동해야해

## 해결 방향

### **해결하고 싶은 문제**

- Smp를 수작업으로 확인하고, LNG발전 가동경제성을 그때그때마다 계산하고 현장에 수작업으로 가동계획 전파하는 프로세스

가동계획 전파방안 : 이메일, 카톡

이메일에는 주간, 야간 LNG발전 최적가동방안, 그리고 스트림릿 페이지에 들어있는 smp, LNG발전 BEP, LNG가격이 표시된 그래프가들어갔으면 좋겠어
[이메일 포맷]
1. 0월0일 가동계획
  - 야간
  - 0월0일+1일 주간[08시~22시]
2. 금일 SMP 및 LNG발전 BEP, LNG가격
   (환율은 밑에 주석으로)

[카톡메시지 포맷]
안녕하십니까, 0월0일 야간~0월0일+1일 LNG발전 가동계획 안내드립니다.
[0월0일 22시~0월0일+1일 08시]
 - 00~00시 : 1기 or 2기 full or 2기 half
[0월0일+1일 주간(08~22시)]
 - 00~00시 : 1기 or 2기 full or 2기 half

추가 안내드릴사항이 있으면 연락드리겠습니다.
문의사항이 있으시면 연락주시기바랍니다.
감사합니다!


### **과제 목표**

- SMP 자동 탐색으로 이상치 구간(0또는 170이상 등)탐지, LNG발전 가동 경제성 자동 계산, 현장 자동 전파로 수급담당자 휴먼에러 최소화

 결과는
 08~09시 이렇게 한시간 간격으로
 LNG발전 최적가동 방안, 해당 시간 경제성, 가동했을때 비용,절감금액,
 SMP 이상치 탐지 시 (170원/kWh) 이상일때는 기력발전 BEP 까지 고려하여 기력발전 점화 안내도 필요
 SMP 이상치 탐지 시 (100원/kWh) 아래일때는 LNG발전 가동 줄이는 방향으로 검토 필요


### **핵심 아이디어 / 접근 방법**

- 매일 17시~19시 사이 전력거래소 SMP 자동 크롤링
평일은 17시반이면 다음날 smp가 나오고, 토요일, 일요일, 공휴일이 껴있으면 좀 더 늦게 나옴
그래서 smp데이터를 끌어오는데 만약 해당시간에 데이터가 없으면 30분 간격으로 다시 데이터를 끌어와야함.
- SMP, 수전단가를 활용한 대체단가 계산
- SMP 이상치 구간 탐색 및 이상치 구간 특히 강조
- LNG발전 경제성 자동 계산으로 경제성 판단
- 메일링 자동화 또는 HMI자동알람을 통한 LNG발전 가동가이던스 제공

### **기대효과**

- 에너지 수급담당자 수작업 최소화로 휴먼에러 최소화
- Smp 이상치 자동탐지를 통한 발전기 경제적 탄력운영 가능
